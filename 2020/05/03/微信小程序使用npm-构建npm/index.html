微信小程序使用npm,构建npm

撰写于 2020-05-03 修改于 2020-05-03 分类 微信小程序 标签 微信小程序

按照微信官方文档中的步骤操作就出出现了下图的错误：

解决方案：

1.进入小程序根目录

2.打开cmd

3.输入：npm init

4.输入命令之后一直点回车

5.输入命令：npm i miniprogram-sm-crypto –production

6.执行命令完之后，点击微信开发者中 工具->构建npm 就会显示完成构建
