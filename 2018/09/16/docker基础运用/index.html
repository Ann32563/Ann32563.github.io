docker基础运用

撰写于 2018-09-16 修改于 2018-09-16 分类 docker 标签 docker

1.查看所有镜像
docker images
2.查看所有容器（包括停止的）
docker container ls -a
3.通过dockerfile创建镜像
docker build -t 镜像名:v1 .
"""
镜像名：v1，v1代表版本号，可以不写
"""
4.运行容器
docker run --restart=always -it -d --name 容器名 -p 8000:80 -v  /home/docker:/docker/code 镜像名

"""
--restart自动重启
-p 8000宿主机:80(docker)
-d后台运行
-v挂载卷
"""
5.进入容器
docker exec 容器ID /bin/bash
6.停止容器
docker stop
7.删除容器
docker rm
8.重启容器
docker restart
9.删除镜像
docker rmi
