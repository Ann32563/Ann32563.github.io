创建博客

撰写于 2018-02-17 修改于 2018-07-17 分类 创建博客 标签 创建博客

1.搭建环境

所有软件都可以去官网下载

1.安装Node.js

安装的时候保持默认的设置即可

检查版本：
在cmd下输入如下的代码，即可查询到安装的版本号

node -v
npm -v

2.安装git

安装的时候保持默认的设置即可

检查版本：
在命令行下输入如下的代码，即可查询到安装的版本号

git -version

3.github账户的注册和配置
1.注册github账号

在github官网注册账号

2.创建代码库New repository

· 仓库名后面要加 .github.io

4.安装hexo
1.创建一个文件夹

文件夹用来存放你博客的资源

2.安装hexo及其它组件

同时按下windows+r输入cmd，在命令行窗口进入这个目录。或者在这个目录下shift+鼠标右键，选择在此打开命令窗口。
在命令行输入

npm install hexo-cli -g
npm install hexo –save
hexo -v

3.初始化hexo

hexo init
npm install
npm install hexo-deployer-git –save

4.生成文件

hexo g

5.启动服务器

hexo s

在浏览器中打开http://localhost:4000/，你将会看到hexo的页面。

2.联系hexo与github page

1.配置git个人信息

之前配置过可以忽略这个步骤，在git bash中输入以下代码：

git config –global user.name “你的github用户名” #配置用户信息
git config –global user.email “你注册github的邮箱”
ssh-keygen -t rsa -C “邮箱” #生成密钥

1.添加公有密钥

记住生成密钥.ssh的目录，登录github官网，在settings的SSH and GPG keys中添加ssh keys，就是你刚刚生成的id_rsa.pub，将末尾的邮箱去掉。

2.测试密钥是否通过

在git bash 中输入命令

ssh -T git@github.com

密钥通过了的话，github官网的添加ssh keys 那里，你刚刚添加的密钥会变绿。

2.配置deployment

按如下修改你在安装hexo时创建的文件夹的根目录下的_config.yml文件：

deploy:
type: git
repo: git@github.com:你的用户名/你的用户名.github.io.git
branch: master

3.配置博客的主题

1.主题安装

去https://hexo.io/themes/找主题，在Git Base中输入相对应的主题的github地址。我的博客使用的是random主题。在Git Base中输入命令

git clone https://github.com/stiekel/hexo-theme-random.git themes/random

2.使用主题

修改你在安装hexo时创建的文件夹的根目录下的_config.yml文件，将主题改为你下载的，例如我的：

theme: random

3.设置主题

一般在你下载的主题的github项目会有主题的设置文档。
我的主题的设置文档如下：

1.头像设置

可以在 Hexo 主配置文件，或者主题的配置文件 theme/random/_config.yml ，修改 avatar 配置，如下：

avatar: /miao.jpg

2.菜单与导航栏的设置

可以在主题的配置文件（theme/random/_config.yml）中自定义。如果想链接到标签和分类的列表页，还需要自己手动创建对应页面。

menu:
首页: /
归档: /archives
标签: /tags
目录: /categories
关于我: /about

执行如下命令会生成一个source/tags/index.md 文件

hexo new page tags #创建新的菜单页面

编辑文件，修改文件的 type 项为 tags，如下：

title: 标签
date: 2016-01-16 06:17:29
type: “tags”
comments: false

3.社交网站连接配置

需要在主题的配置文件 theme/random/_config.yml 中添加对应的配置，如下：

social:
GitHub: https://github.com/Ann32563
Weibo: https://weibo.com/u/5265686023
zhihu: https://www.zhihu.com/people/xiao-ya-jing-94/activities
csdn: http://blog.csdn.net/qq_40072810
blog: http://www.cnblogs.com/Ann32563/

4.第三方服务配置

本主题支持 Google Analytics / 百度统计 / Disqus / 友言 / JiaThis，即可以在主题配置文件中设置，也可以在博客的主配置文件中设置，增加项目如下：

Google analytics

google_analytics:

Baidu tongji

baidu_analytics:

Disqus

disqus_shortname:

Uyan

uyan_id:

jiaThis

jiaThis_uid:
jiathisConfig:
data_track_clickback: true
summary: “”
showClose: true
shortUrl: false
hideMore: false

也可忽略特定服务器的访问统计

isIgnoreSpecialHostVisitor: true
ignoreHostList:

- localhost
- 127.0.0.1
5.hide按钮

在每个页面的左下角，都有一个 Hide 按钮，点击这个按钮之后，按钮之外的东西都会被隐藏。如果不想显示这个按钮，可以在主题或者 Hexo 的主配置文件中修改 hideButtonVisibility，设置为 false：

hideButtonVisibility: false

6.背景图设置

使用 unsplash 提供的随机图片。

unsplashConfig:

# 关于这几个配置项的说明，请参见 https://unsplash.it/
# blur: true # 给图片增加模糊特效
gravity: 'north' # north, east, south, west, center # 图片翻转
# greyscale: true # 使用灰度图片

可以自己配置背景图片，也是在主题配置文件中进行，URL 中的 width 和 height ，在请求时，会替换为浏览器的宽度和高度值，如果你的图片服务器支持按分辨率裁剪，那就可以用得上。如下：

backgroundImages:

  • http://7te9fe.com1.z0.glb.clouddn.com/bgimg_1.jpg?imageView2/1/w/__width__/h/__height__
  • http://7te9fe.com1.z0.glb.clouddn.com/bgimg_2.jpg?imageView2/1/w/__width__/h/__height__
  • http://7te9fe.com1.z0.glb.clouddn.com/bgimg_3.jpg?imageView2/1/w/__width__/h/__height__
  • http://7te9fe.com1.z0.glb.clouddn.com/bgimg_4.jpg?
    7.背景图轮播设置

vegasConfig:

# animation: 'random'
preloadImage: true # 图片预加载配置
transition: # 图片的切换物资
  - slideLeft2
  - slideRight2
  # - zoomIn
  # - swirLeft
  # - swirRight
  # - flash
  - flash2
# transition: 'swirlRight' # 可以只配置一个
timer: true # 是否显示时间进度条
delay: 5000 # 每张图片的显示时间
shuffle: true # 按顺序播放，还是随机播放
count: 10 #一共多少张图片

关闭背景图轮播

turnoffBackgroundImage: 

# home page
index: false
# arcihve / categories / tags
post_list: false
# post and page
post: true

关闭背景图后，可以通过下面的变量来设置背景色，注意，不要包括 # ：

backgroundColor: D7CCC8

4.写博客、发布文章

1.新建一篇博客

在hexo的目录下打开命令行

hexo new post “你文章的标题”

这时候你再hexo目录下的\source\ _posts 将会看到 你文章的标题.md 文件，用markdown编写文章

2.部署上线

hexo g
hexo d

或者可以一步完成生成和部署

hexo d -g

部署成功后访问 你的地址，https://你的用户名.github.io
