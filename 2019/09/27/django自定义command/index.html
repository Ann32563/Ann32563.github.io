django自定义command

撰写于 2019-09-27 修改于 2019-10-26 分类 Django 标签 Django

1.创建目录

在app内创建一个management的文件夹，在此文件夹里新建commands文件夹，在此文件夹中创建py文件。

2.执行

py文件名就是我们自定义的命令，执行：

python manage.py report_all(命令名)

3.新建命令

在命令文件中，需要定义一个Command类并继承BaseCommand。
handle()是主处理函数，add_arguments()是接收参数的。

执行：

python manage.py report_all '2019-9'
