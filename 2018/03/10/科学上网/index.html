科学上网

撰写于 2018-03-10 修改于 2018-09-17 分类 科学上网 标签 科学上网

使用的是shadowsocket实现翻墙

1.注册

进入亚马逊官网，注册亚马逊账号，填写个人信息，需绑定一张信用卡（淘宝虚拟信用卡），接下来会有一个验证码，会打电话过来，输入验证码，就可以成功激活了。

2，创建实例

登录aws主页，点击我的账户，右上角用户名边上有一个地区选择，根据自己的需求选择一个位置创建主机，点击启动实例，选择一个Linux系统，推荐ubun，记得选择的时候选带有符合条件的免费套餐，之后点击启动，会让你创建密钥，输入一个名字，之后下载存储秘钥，启动实例，就可以在你的服务器中看到了，服务器ip记录下来。

  • 免费云服务也是有限制的，每月750小时（24*31），每月100w次请求，只能创建一个实例，不然会扣费！
3，连接实例

点击启动实例后面的连接，里面会有多种连接方式，推荐大家使用ssh连接。使用ssh连接需要使用之前生成的秘钥，窗口出现ubuntu@x-x-x表示连接成功

4.安装软件

提升权限，安装软件

sudo 进入root用户
apt update   更新软件源
apt install python-pip    安装python包管理工具
pip install shadowsocks   安装上网软件
5.配置软件

编写一个配置文件，在/etc/shadowsocks.json中编写配置（如果没有就创建）,
单用户配置:
{
“server”:”0.0.0.0”,
“server_port”:”9229”,
“local_address”:”127.0.0.1”,
“local_port”:”1080”,
“password”:”自己的密码”,
“timeout”:500,
“method”:”aes-256-cfb”,
“fast_open”:false
}

多用户配置:
{
“server”:”0.0.0.0”,
“server_port”:”9229”,
“local_address”:”127.0.0.1”,
“local_port”:”1080”,
“port_password”:{
“9229”:”密码”,
“9230”:”密码”,
“9231”:”密码”
}
“timeout”:500,
“method”:”aes-256-cfb”,
“fast_open”:false
}
建议其实单个用户的配置足够，一个账户也可以多个设备登录

6.启动服务器

在终端中启动服务器输入以下命令，加入开机自启，

ssserver -c /etc/shadowsocks.json -d start

编辑/etc/rc.local，启动ss

sslocal -c /etc/shadowsocks.json

去服务器配置中打开端口，到自己的实例上。找到安全组，进入安全组设置，点击入站，编辑，点击添加规则，在端口范围里填上我们设置的端口9229，来源下拉框中选择任何位置。

7.连接使用

在Android、mac、linux、windows上可以使用shadowsocks客户端，输入相关账号信息即可。
